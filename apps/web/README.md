# ACME Web App

Astro app for ACME.

Used as a hybrid framework (both backend and frontend) to generate static web pages.

This app uses:

- [CMS Client](../../packages/clients/cms)

## Setup

```bash
pnpm install
```

## Available scripts

### Run

```bash
pnpm dev
```

### Preview built app

```bash
pnpm preview
```

### Build

```bash
pnpm build
```
